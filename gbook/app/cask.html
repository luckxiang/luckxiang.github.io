Homebrew Cask

Homebrew的扩展，和 Homebrew 使用方法差不多，可以安装和管理 OS X 图形界面程序。我一般优先在App Store 或官方下载。

安装
brew install caskroom/cask/brew-cask
使用

安装一个程序的命令如下，其他参考Homebrew。

brew cask install FORMULA
